# Ticket Importer

Small Textual app for preparing, previewing, and submitting bulk ticket imports.

Run it from the repo root:

```bash
source functions/.venv/bin/activate
python functions/.scripts/ticket_importer/ticket_importer.py
```

If you still need to install the environment first, follow the full guide in:

- [Ticket Importer Documentation](/Volumes/PhD_SSD/Projects/the-engine/apps/tools/documentation/docs/docs/ticket-importer.md)
